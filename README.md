# ⚡ 데바니온 최적화 시뮬레이터 (Daevanion Build Optimizer)

포인트 제한 내에서 사용자가 원하는 우선순위 스탯과 고유 스킬 활성화 경로를 실시간 최단 거리로 계산하는 시뮬레이터입니다.

---

## ✨ 주요 특징 (Features)

*   **자동 빌드 추천**: 보유 포인트를 기반으로 최적의 노드 연결 경로를 실시간 연산하여 활성화합니다.
*   **우선순위 커스텀**: 우측 패널에서 스탯 카드를 드래그하여 가중치 순서를 자유롭게 변경할 수 있습니다.
*   **인게임 규칙 완벽 반영**: 데바니온 보드(네자칸/지켈/바이젤/트리니엘)는 공통 포인트를 공유하며, 신성 보드(아리엘/아스펠/마르쿠탄/유스티엘)는 단독 예산을 사용합니다.

---

## ⚡ 핵심 최적화 (Optimizations)

*   **다익스트라 캐싱(Memoization)**: 보드 상태를 해싱하여 중복 다익스트라 연산을 차단, 최적화 속도를 5~10배 단축했습니다.
*   **DOM 재활용 (DOM Recycling)**: 격자판 노드 225개를 재생성하지 않고 1회 렌더링 후 스타일과 이미지만 갱신하여 렌더링 성능을 20배 향상시켰습니다.
*   **보안 및 자원 절약**: XSS 방어(`textContent` 사용), BFS 성능 개선, 대용량 데이터 로드 캐싱 버전화(`?v=1.2.0`)를 적용했습니다.

## 📂 프로젝트 구조 (Structure)

```bash
├── assets/                          # 격자판 배경 및 비주얼 이미지 리소스
├── data/                            # 15x15 데바니온 배치 정보 및 스킬 매핑 데이터
├── index.html                       # 메인 마크업
├── index.css                        # 스타일시트 및 네온 이펙트
├── index.js                         # Dijkstra 코어 연산 및 인터랙션 스크립트
└── README.md                        # 실행 가이드 문서
```

---

## ⚙️ 실행 방법 (How to Run)

로컬 보안 정책(CORS) 우회를 위해 로컬 서버 환경에서 실행해야 합니다.

```bash
# 프로젝트 루트에서 로컬 웹 서버 실행
python -m http.server 8000
```
웹 브라우저를 열고 `http://localhost:8000` 주소로 접속하면 즉시 실행됩니다.
